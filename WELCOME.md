# Welcome to TDT Space! 👋

Thank you for checking out TDT Space - Multi-Agent Terminal for TDT Vibe Coding!

## 🚀 Quick Start

### Installation
1. Download the latest release from [Releases](https://github.com/TheDemonTuan/all-agent-in-one/releases)
2. Extract the ZIP file
3. Run `TDT Space.exe`

### First Steps
- Create your first workspace
- Choose a layout (2x1 or 2x2 recommended for beginners)
- Start coding with your favorite AI agents!

- 💾 Workspace management
- 🎨 Custom templates
- 🔍 Terminal search
- ⌨️ Command history
